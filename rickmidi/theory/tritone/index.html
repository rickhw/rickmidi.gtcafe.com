三全音 (Tritone)

三全音在音程具備極具張力的聲響特性，了解基本概念與應用方法，對於情緒的營造與音樂表達會很有幫助。

三全音是指音程是 三個全音 的距離 (六個半音)，也就是 增四度 (augmented fourth) 或 減五度 (diminished fifth)。例如：C - F# 或者說 C - Gb 的距離。這個音程的聲響特性就是極度的不安定，詭異。有人用魔鬼、地獄、痛苦、銷魂 … 來形容。

基本概念

最常出現三全音的是大調順階七和弦的五級。C 大調的五級和弦 G7：G B D F 中的 B - F 就是一個 增四度，同時也是造成屬七和弦聲響不穩定的主要原因。所以一般來說，因為聽到屬七和弦的不穩定性，使得聽覺心理會趨向、渴望聽到穩定的感覺，調性音樂中最穩定的就是 一級。從不穩定的轉向穩定的過程，在音樂術語上稱為 解決 (Resolved)。

因為只有屬七和弦才會有三全音，所以意味著，一個大調音階裡，只會有一組三全音。所以我們可以根據一組三全音來推出該組三全音屬於哪一個調。例如 B - F 的距離，把 B 加一個半音，可以推出是屬於 C 大調的。

另外，三全音剛好把一個八度音切成兩半，所以 C - F#、F# - C 都是三全音。所以上段描述的 B - F 反過來 F - B 也是三全音，把 F 加一個半音可以得知是 F# 大調的三全音。也就是說，同一組三全音會分別屬於 C 大調和 F# 大調，可以說是一體兩面，從不同角度看一個東西.

Comping (伴奏) 的時候，使用 Shell Chord (根音 + 三度音 + 七度音) 的基本概念就是強調出和弦屬性的 特性聲響 (Guide Tone)，屬七和弦的 Guide Tone，G7 也就是 B - F：三全音。所以 Comping 屬七和弦選擇彈奏 Guide Tone 用意就是要強調出三全音的聲響。當然其他七和弦也是要強調出她們各自的聲響.

判斷三全音

要找出三全音的音程，可以利用五度圈來找。三全音就是增四度或者減五度，所以五度圈順時針的方向 F - C - G - D - A - E - B，因為每一組都是完全五度的關係，所以每一組當中只要五度音少了半音，那麼就是一組三全音的關係，例如:

  • F - C => F - Cb(=B)
  • C - G => C - Gb
  • G - D => G - Db
  • … 類推

同理，五度圈逆時針方向是完全四度，C - F - Bb - Eb - Ab - Db，只要四度音增加一個半音，也會形成一組三全音:

  • C - F => C -F#
  • F - Bb => F - B
  • Bb - Eb => Bb - E
  • … 類推

降二代五 (Tritone Substitute)

另外三全音的應用最有名的就是 降二代五 (Tritone Substitute，三全音代理)。爵士樂理最常用的 ii-Ⅴ-I 進行中的五級，以 C 大調來說:

Dm7 - G7 - Cmaj7

改成：

Dm7 - Db7 - Cmaj7

其中和弦的組成：

  • G：G B D F => B - F
  • Db：Db F Ab Cb(=B) => F - B

他們都有一組三全音，同時 Bass line 由原來的 D - G - C 變成 D - Db - C，經過音。這樣的代換，除了保留了原本三全音的聲響，同時又讓 Bass Line 增加了 Passing Tone (經過 音) 的效果。另外就是 G 和 Db 和弦的組成音，上下比較，除了 B - F 之外，G - Db 是三全音，D - Ab 也是三全音，很神奇吧 ~

減七和弦 (Diminshed 7th)

另外一個跟三全音有關的是 減七和弦 (Diminshed 7th)。減和弦把八度音平分四等份，以 Cdim7 來說:

Cdim7：C Eb Gb Bbb(=A) = Ebdim7 = Gbdim7 = Adim7

三全音是把八度音分成兩等份，所以以數學概念來看，12 個半音對半切可以找到三全音; 切成四份，可以得到減七和弦，所以不難發現減七和弦會包含兩組三全音:

Cdim7 => C - Gb 和 Eb - Bbb

這也是單獨彈奏減七和 弦的琶音 (Arpeggios) 時，會有更魔鬼的感覺.

結論

三全音在現代音樂中，應用的非常廣泛，除了上述提到的基本概念和和聲代換技巧，另外一個就是藍調。都是使用屬七和弦和弦進行的藍調裡，三全音聲響出現的頻率就更高了。
