    Rust 的版本管理工具

    Hello Rust

    第一個 rust: main.rs

    fn main() {
    println!("Hello, world!");
}

    解釋：

    1. 檔案名稱
      • Rust 使用 .rs 當作副檔名
      • 檔案名稱如果有多個單字, 建議使用底線, 像是 hello_world.rs 而不是 helloworld.rs
    2. 函式:
      • Rust 預設的 function 名稱叫做 main, 這個名稱是特殊的名稱.
      • 函數使用 () 作為參數傳遞方式
      • 使用 {} 作為範圍區隔
    3. 使用四個空格作為縮排的排版風格 (First, Rust style is to indent with four spaces, not a tab.)
    4. println! 表示 rust macro, 如果想要調用 function, 則把 ! 刪除.
    5. Rust 使用分號 ; 作結束.

    編譯 main.rs 並執行:

    $ ls -l
total 8
-rw-r--r--  1 rickhwang  staff  45 Feb 22 17:26 main.rs

$ rustc main.rs

$ ls -l
total 792
-rwxr-xr-x  1 rickhwang  staff  401208 Feb 22 18:00 main
-rw-r--r--  1 rickhwang  staff      45 Feb 22 17:26 main.rs

$ ./main
Hello, world!

    嘗試讓他 compile error 看看，先刪掉 ;

    fn main() {
    println!("Hello, world!")
}

    compile 過了 ...

    $ rustc main.rs

    刪掉一個 }:

    fn main() {
    println!("Hello, world!")

    編譯訊息：

    ❯ rustc main.rs
error: this file contains an unclosed delimiter
 --> main.rs:3:2
  |
1 | fn main() {
  |           - unclosed delimiter
2 |     println!("Hello, world!")
3 |
  |  ^

error: aborting due to previous error

    改成這樣：

    fn main() {
    println!("Hello, world!")
    println!("Hello, world!");
}

    出現的訊息，算很清楚：

    ❯ rustc main.rs

 --> main.rs:2:30
  |
