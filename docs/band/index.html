怎樣的程度才適合組樂團？

這種東西會有點主觀，所以我個人大概分成兩個目的好了：娛樂性 和 職業性

關於寫譜

我寫譜大部份都是給自己看的，因為寫譜需要極高的專注力、聽力、節奏感、樂理、排版美感、專業工具等知識，是件非常非常耗費精神的工作 (不會輸給寫程式)。

樂團的經營

玩 Band 是很棒的事情，一群同好在練團室裡一起享受自己彈奏／演唱的音樂，實為人生一快！不過要能夠 enjoy 這一快，也要做一些基本的功課，否則也有可能會變成一種『折磨』。
