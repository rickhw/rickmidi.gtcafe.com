樂團的經營

經營樂團跟經營公司一樣，怎處理人、事、物，都是學問。

玩 Band 是很棒的事情，一群同好在練團室裡一起享受自己彈奏／演唱的音樂，實為人生一快！不過要能夠 enjoy 這一快，也要做一些基本的功課，否則也有可能會變成一種『折磨』。

以下是一些我覺得可以留意的『人事物』。一些是來自於前輩的心得、經驗，有一些是我個人的體會和經驗。主要目的都是讓『練團』這件事情，除了是一件愉快的事情，同時也是讓團員之間能藉由練團的互動，培養默契、踏實經營、最後還有音樂的成長。

《人》

『人』是最重要的、『人』的部份對了，樂團就成功一半了。這句話跟老闆找員工的意思是一樣的；跟找到心靈契合的伴侶，人生已完美一半類似。

要找到四五個人都同樣喜歡同類型的東西、程度相仿、對於音樂的態度一致、個性上又能夠溝通、能夠在同時間、同樣的地點一起練習，其實是不容易的。

其中我個人最在意對於練團的態度。態度不佳，很快就會 bye bye！

程度不好沒關係，肯學、有練習就會慢慢進步。現在網路資源很豐富，家教也到處都有，技術的東西，大部分花時間投資練習，只要方法得宜，就會慢慢看出成果，只要不眼高手低。

已經有程度的人，卻又不肯花時間採譜、也沒有練習，到練團室才在現場採譜的，不然就還在問今天要練哪些歌的？或者只想玩個人秀，不懂得收放，都是讓人搖頭的，也可能很快的會被請出去。因為，千萬別忘了，『樂團』是一個團體。

我個人覺得有程度的人，應該是像戰場上的將軍一樣 (註)，能夠懂得何時該收、何時該放，知道音樂的起伏、深度、廣度，然後拿捏適當的分寸，這才是比較正確的態度。如果只是一味想要個人秀，其實只會顯示自己的不足。

註：三國演義 一百回：漢兵劫寨破曹真，武侯鬥陣辱仲達

諸葛亮致書曹真：『竊夫為將者：能去能就，能柔能剛；能進能退，能弱能強。不動如山岳，難知如陰陽；無窮如天地，充實如太倉；浩渺如四海，眩曜如三光。預知天文之旱澇，先識地理之平康。察陣勢之期會，揣敵人之短長。』

樂團是幾個有不同專長的人一起做音樂，所以對於溝通的方式就顯得很重要。現在網路發達，各式各樣的通訊介面：IM (Facebook Messager、Line … etc)、BBS、Blog、Wiki、群組論壇 … 各式各樣都是可以作為討論樂團音樂的地方。善用這些介面，溝通關於人事、音樂總譜、活動訊息、公關 … 各項細節等，跨時間、空間，只要有心，可以讓練團更順利。

而上述的平台介面，很多系統都有隱密性、檔案管理、Rich Text 格式 … 等，用來放音樂資訊 (總譜、討論樂理) 是再適合不過了，而且大部分都不用錢。以現在這個資訊流通很快年代來說，說訊息沒接收到其實都不是理由。

剩下的是屬於基本的做人處世道理：守時、不爽約、認真負責 …. )(*&^%$#@……. 這些就不提了。

上述的人事部分，大部分由樂團的團長或者樂團中擅長溝通者來負責。

《事》

簡單說就是跟音樂有關的東西都屬於『事』。其中最主要的就是用來練團時用來『溝通』的團譜、總譜。在談總譜、編制、編曲之前，先瞭解一下專業的是怎麼做這些事情的，有那些可以參考的。

一、大型演唱會

在一般演唱會有一個角色叫做音樂總監。他主要工作，就是提供統一、正確的歌曲總譜給每個樂手，同時也要負責音樂各個聲部的配器安排和樂手的溝通。

例如原曲如果是 Orchestra 編制，但是實際預算並沒有請管絃樂團，那麼 KB 要如何安排？或者需要改成小編制的絃樂組？或者兩組 KB，一組負責 Pad，一組負責 Comping、Effect？

或者原曲是三把木吉他的編曲，但是只有一個吉他手，那麼需要做怎樣的調整和安排？有三個鍵盤手，或者三個吉他手，要如何分配、區分他們的職責？

改編的歌曲，是否需要多增加 Percussionist？減輕鼓手的負擔？沒有預算要怎麼做？Loop／Sampling？歌曲進行當中有穿插臨時樂手，像是古典吉他、Saxophone、國樂 …. 要怎麼安排他們的位置和收音、監聽？

是否需要與燈光、舞台效果（一般是煙火爆破的視覺效果、類似 101 跨年煙火那種有 Timing 的特效）、投影畫面、現場連線等 … 配合，同時由鼓手經由打擊版送同步 MIDI 訊號？… 等。

二、樂團的總譜

一般的樂團裡相對比較單純，但是還是會遇到最常見的問題：譜的正確性與配器問題 (特別是 GT vs KB 相互支援)。

樂團裡最好有一個樂理、節奏感、聽力都比較 Ok 的人來負責整理總譜，通常這個人會是 KB or GT，其他團員有義務協助更正、和給予適度的建議。

例如比較複雜的節奏，鼓手可以提供他的經驗給大家參考；Bass 可以提供比較漂亮的 Bass Line 來呼應鍵盤或吉他的和弦。鍵盤手可以讓大家多認識一些 GM 音色或者其他特別的 Loop / Sample、吉他手可以告訴大家混音 (聲音頻率) 的一些概念。這也是團員們彼此交流音樂 / 互相學習的最好機會。

總譜包含幾個重要的訊息：段落、和弦 (級數)、插音 (Fill)、節奏、速度、調性等。資訊越簡潔越好，千萬不要把 Guitar Pro 那種六線譜帶去練團室翻，那是在家就要做好的功課。

總譜的重要性以及相關的排版工具、寫法，請參考『樂團的總譜』的介紹。

三、段落

總譜還有另外一個很重要的就是段落。如果是 Cover 團，經常會遇到原曲是 Fade out (淡出) 的歌曲。這時候大家要討論一下，該怎麼結束。可以自己改編，可以參考原始樂團的 Live 演出，Youtube 是很棒的媒介。

總譜上面，段落的標示要最清楚，通常習慣用英文的名詞標示：

  1. Intro (前奏)
  2. Verse (主歌)
  3. Pre-Chorus (預備段落)
  4. Chorus (副歌)
  5. Bridge (發展部、橋段)
  6. Chorus' (主題重現、副歌重現)
  7. Outro (尾奏) …. 等。

樂手的譜要跟歌手的歌詞對應，如此在練團時溝通才不會雞同鴨講。不然在溝通段落就會浪費很多時間。

更多關於段落的說明, 請參考 “怎樣練一首歌”.

同時要注意段落之間的變化，同樣是 Verse 或者 Chorus，編曲者通常不會讓第一次和第二次一模一樣，兩段編的一樣是很遜的。所以聽歌的時候請注意這些細節，因為一個團有沒一定水準，有的時候這些細節是看的出來的。

另外總譜上面也會標示除了主旋律以外的音符，像是插音 (Fill) 、特別的節拍 … 等，通常也會著名音色，常見的縮寫：

  1. PN: piano
  2. EP: Electric Piano
  3. GT: Guitar
  4. EG: Electric GT
  5. AG: Acoustic GT
  6. STR: String
  7. BS: Bass
  8. DR: Drums
  9. DB: Double Bass
  10. VLN: Violin
  11. …. etc

總譜最好以一張 A4 為原則，最多兩張。因為現場表演時，大家手都很忙，沒時間翻頁的。

在總譜的整理，除了基本的音樂資訊 (男女的 Key、Tempo) ，還可以把一些比較特別的和聲技巧勾勒出來，在練團前利用討論區討論。例如歌曲中用到調式、利用到 Modal Interchange、用到了少見的減七和弦、用到了平行調、關係大小調 … 等音樂技巧。如果大家都瞭解了，當然就不用了。

負責總譜的人，除了上述以外，對於其他樂器和音樂的術語，例如鼓的一些基本概念、Bass or GT 效果器的概念、 MIDI GM 的音色、還有就是節奏的表達、速度的概念、曲風的輪廓、常見曲式 … 等，要可以用清楚的文字或語言和其他人溝通。

因為鼓手不一定會知道什麼叫做 1 - 2m - 3m -4M7 的和弦進行；鍵盤手也不見得知道吉他手的 C 要怎麼按、他的轉位和鍵盤有啥差異；大部分的人可能不瞭解 KB 的分鍵原理、還有分鍵的演奏難度；大部分的人可能聽不出來 Soft Pad 和 EP 有啥不一樣？也不太知道 Synth Lead 的聲音是怎麼一回事；大部分的團員可能不知道吉他 OD (Overdrive) 和 DS (Distortion) 有啥不一樣 …

但是音樂術語大家通常都會懂，練團的時候就可以拿來溝通。

※更多關於吉他轉位與和弦聲位, 請參考 “吉他和弦聲位的基本概念” 說明.

四、樂團形式

Cover 團

簡單說就是 Cover 已經上市發表的音樂。配器的安排要依據樂團編制來設計，特別是 Cover 歌曲的樂團，可能常常會因為要 Cover 的歌曲和樂團成員編制有所落差，而需要做調整、甚至編曲。三把吉他沒有 KB、KB 要怎麼協助？ Ex: Big City Night. 或者是沒有吉他、卻有一堆 EP, 弦樂 (ex: Carpenters - Close to you) 時，吉他手能否幫忙？

創作團

主唱最好也懂一些基本的樂理概念和樂器。通常會是木吉他。創作團要花很多時間在討論編曲，基本上寫譜更是重要於 Cover 團。討論的結果事後要有人去整理成總譜，草草了事，之後如果遺忘了會很可惜的。

雙吉他 / 雙 KB 編製

一個團裡有兩把以上的吉他，那麼吉他手們要分配好工作，誰彈 Riff, Rhythm、誰彈 Lead、Fill 。除非聲音不夠滿，否則盡量兩個人不要彈同樣的東西。即使都要彈 Chord，也要彈不同把位的、或者用不一樣的音色 (ex: 一把選 Marshall、一把選 Mesa)、適度的放空和彈滿，如此音樂聽起來才會有層次，而不是從頭到尾的情緒都是平的。

《物》

譜、練團室、設備、硬體、樂器等都是物。

練團前要做什麼準備？

簡單說要練習、採譜、確定歌曲的版本，還要思考是否需要改編。通常和聲樂器 - GT 跟 KB 要先溝通好。是否有有比較特別的東西要事前準備？像是 Loop？歌曲是否需要？誰來放 Loop？怎麼監聽 (這是大問題, 可以看看這些討論 - “鼓手與 PA 的關係”, “表演時 Loop 怎麼放” 當代樂手)？遇到 Fade Out 的要怎麼辦？遇到有戲劇效果的要怎麼辦？遇到國樂／民俗編曲怎麼辦？

※ Loop 的概念與應用請參閱 “Loop 的概念與應用實例 - One Night in 北京” 的介紹

如果歌曲有很多種版本（原版、不插電版、現場版、搖滾版、絃樂版、吉他版、爵士版 … ），最好確定一下是哪一個版本，團長 or 音樂總監應該提供統一的版本給所有團員，所有團員不要又跑去 youtube 找其他版本。甚至音樂總監要重新編一個版本給所有團員。

如果是原曲吉他降半音的，可以考慮是否大家跟著降半音，同時以後也都用降半音來做，或者吉他要思考一下該如何改編？原本是 C 大調的曲子，改成 B 大調，但是彈出來的聲響 Vocing 要跟原曲一樣，要安排一下。像是 GNR 的曲子。

歌曲降半音在音樂圈是很常見的慣例，很多樂團都習慣這樣做，特別是搖滾歌曲。所以大家要先有這種 Sense，練團前先溝通好，培養好默契，之後練團就是以 B 調作為預設的調性作調整，升一個半音就是實際上的 C 調。Keyboard 可以直接把 Transport 設定好，存起來，這樣以後練團才不會弄錯。

如果主唱音域唱不上去，請主唱要能夠適度告訴大家他的音域。主唱要知道自己的音域範圍，會一點樂器跟簡單的轉調概念是最好的方法，如此方便跟其他樂手溝通。找出適合自己的音域範圍，然後才能盡興的詮釋歌曲，這樣唱起來的人不會覺得難唱，聽的人也不會覺得痛苦。

※ 關於主唱的說明，請參考『樂團的主唱』

另外就是重視練習的品質，我個人遇到比較難的曲子時，練團前會自己把歌曲做好 Backing，然後在家自己練習。這樣在練習時等於已經在練團了，到練團室就不是要去修正什麼演奏技巧、或者和弦的問題，而是如何去利用練團室器材表現音樂、和團員培養默契的時候。

最後很重要的，要有人去整理總譜，然後所有人要去幫忙檢查和弦、段落、節奏、速度、調性正確與否、是否需要重新安排吉他和鍵盤編制 / 分配。吉他跟鍵盤常會需要這樣互相幫忙，所以不管樂團是兩把吉他、還是吉他鍵盤各一個，請在進練團室之前把這些事輕溝通好。這些事情不要進了練團室再討論，因為那都是浪費時間、浪費金錢的。

在練團室裡要做些什麼？

租練團室練習我個人覺得最重要的就是：善用裡面的器材、培養默契的時候。在練團事理討論和弦、段落、編曲 … 都是浪費錢、浪費時間的。那些東西大部分都因該在練團前就要搞定的。

到練團室就是去玩那些 Amp、設備，去感覺 Balance、吉他手甚至要去把每次遇到的音箱型號、Settings 記下來。其他樂手也是一樣要注意 Balance，尤其是鼓手。Punch 要多大？不同大小的練團室、不同吸音品質的練團室，都需要做調整。

開始練歌之前，大家要測試一下 Balance。跑一段簡單的節奏、和弦進行 (Ex: E Blues)，在跑的過程當中每個樂手依序修飾自己的 Balance。同時這時候可以培養默契，大家輪流 Jam。

歌曲練習中，如果有遇到段落還是有問題的，要改編的，這時候之前準備好的總譜請拿出來討論。通常沒有總譜的時候，在那邊放 MP3、找 Youtube、都是雞同鴨講，浪費時間。所以要再說一次：前置作業很重要。

另外可以的話也帶錄音設備去把練習的過程錄起來，事後聽看看可以檢討的部份，聽看看自己彈的根原曲的距離、聽看看哪些細節可以弄得跟漂亮、聽看看音色可以再修正的地方、聽看看哪裡可以再加入 Fill 、聽看看哪裡可以改編 ….

練團時也要注意一下該有的基本素養，進到練團室每個樂手屬性不一樣，但是都會有一些前置作業，例如：

  • 鼓手：調整椅子高度、確定各個打擊點的位置是適合自己的、確定大鼓位置是否有固定，打小鼓、Cymbal 測試 Punch 和該練團室的音場、譜架高度調好。忌諱一進練團是就開始瘋狂輪鼓、狂過門，別忘了練團室還有其他人要 setting。
  • 吉他手：把音箱的音量調好、琴的音準調好、背帶高度調好、譜架高度調好、有麥克風的話也一併調整好。調整好音箱之後，有必要的話請把音箱的 Settings，包含音箱型號、EQ 的參數記下來，但是請不要在練團是花時間調 Tone，上述的東西在三分鐘之內完成。請不要在練團室練習調音或者測試你的效果器有多麼的會趁、或者你的 pickup 出力有多大、也不要急著要大家面前秀你的 sweep 有多快 …
  • Bassist：跟吉他差不多。
  • Keyboardist：如果是自己的琴，請在進練團室之前就把常用的音色設定好，如果使用練團室提供的琴，那就把常用的音色 (GM) 設定好。練團室琴的廠牌型號，練團前可以打電話問一下，上網找一下說明書大概看看。如果操作很複雜，找到鋼琴、EP、絃樂等常用的音色先頂著用。
  • 主唱：把譜架麥克風架高度調整好，測試 Mic 音量 … 不要打麥克風.

這些要一點經驗也要一點基本的素養，練團的品質與制度會決定台上的表現。如果是比賽，Setting 時就已經開始評分了。

練完團之後要做什麼？

練團室裡沒完成的，除了設備以外的，其他都跟練團前一樣。思考練團時遇到的問題，要怎麼改？怎麼編？上論壇討論。

樂器頻率

練團室中提到很多關於 Balance 的問題。不是聲音開大或者開小就好。很多時候是樂器之間的頻率問題。

大部分的練團室都很小，樂器音量都很大，為了要達到讓每個樂手都能夠聽到彼此的聲音，頻率要做一些分配。一般來說中低頻 (80~160 Hz) 不要開太大，因為這個頻帶很多時候是造成練團室轟轟響的主因，但是因為練團室空間的關係，為了不要折磨彼此的耳朵，所以會建議砍一些。好的練團室也會加強吸音這個頻帶。

鼓手視狀況而定，適度的修飾 Punch、不一定要打很大力。因為通常鼓手打太大力的後遺症就是，其他樂手聽不到自己的聲音，然後大家就會把自己的音量都跟著調大，然後鼓手發現聽不到自己的聲音，又更用力打，如此惡性循環，最後出練團室，大家耳朵都受傷了。
