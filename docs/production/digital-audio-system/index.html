Digital Audio System

為了方便跟朋友解釋現代音樂工作環境的架構，花了一點時間畫了一張簡單的架構圖。

為了方便跟朋友解釋現代音樂工作環境的架構，幾個月前我就花了一點時間畫了一張簡單的架構圖。後來又花了一點時間進一步整理了相關資料，最後彙整出這張數位音樂系統 (Digital Audio Workstation) 的架構圖。

圖中提到很多軟硬體、訊號流…等概念，可以簡單說明現在市面上流行的產品，像 Reason、Sonar、Nuendo、Cubase … 等在這張圖上的角色﹔同時也可以了解現代音樂製作的大概流程。想要架構自己音樂工作環境的朋友，也可以以此圖為基本概念，然後規劃實際的軟硬體架構。

以下解釋圖中每一個部分的名詞，第一部分為訊號流，也就是左下角籃底的部分：

  • MIDI：Musical Instrument Digital Interface，數位音樂樂器的溝通介面。是音樂科技產品的工業標準。可以指的是：硬體介面 (MIDI Interface)、檔案規格 (Standard MIDI Files, SMF)、訊號格式 (MIDI Message)、音源規格標準 (Communicate Spec., ex: GM, XG, GS, GM2 … etc) … 等
  • MIDI Message：MIDI訊號。只是紀錄音符的相關資料，不包含原始聲音。MIDI Message最基本的訊號有：
    • Note number (音符 0~127)
    • Velocity (力度 0~127)
    • Note Off (音符結束時間)
    • Note On (音符開始)
    • Control Change (控制表情參數)
    • Program Change (音色控制碼)
    • Pitch Bend Change (滑音)
  • Digital Audio，數位聲音訊號：將聲波轉化成數字，將這些數字進一步解碼之後，變成只以0 與 1 所表示的一串訊號，立於電腦處理以及儲存。在實際的訊號傳輸過程中，就是以高電位來代表 “1”，以低電位代表 “0”。數位聲音訊號只會存在於數位系統之中，要 ”聽” 到這樣訊號的聲音，必須經過 Audio Interface (音效介面) 作數位類比轉換 (D/A)，方能真正呈現給我們的耳朵聆聽。
  • Analog Audio，類比聲音紀錄訊息：是將聲波化成實際的電壓變化，由訊號導線傳送或儲存在磁帶裝置。如果用示波器去實際偵測類比訊號傳輸中的電壓變化，則會在示波器上看到類似聲波的圖形顯示。

系統名詞

第二部分為系統名詞說明：

  • 黃底：軟體 (低成本，品質不錯，汰換更新速度快)
  • 灰底：硬體 (成本高，較傳統，有所謂的 Vintage 味，較有蒐藏價值)
  • 黃灰相間：軟硬體均有 (看預算 … )
  • DAW (Digital Audio Workstation)：分成軟硬體兩個部分
    • 軟體：指的是能夠處理錄音、修改聲音、播放、混音和處理 MIDI的音樂軟體工作環境。特別是針對 Audio 的處理器、數位非線性剪輯、編輯等運算處裡功能。目前比較著名的 DAW 有 Cubase、Nuendo、SONAR、Logic、Digital Performer、ProTools … 等。
    • 硬體：指的是能夠與軟體 DAW 溝通的機器，像是可以直接透過硬體 DAW 操作 Playback、Stop、Record … 等功能。目前比較新的控制器 (MIDI Controller) 都支援直接操作 DAW 軟體。
  • Controller (控制器)：可以直接送 MIDI Message 和控制參數給 DAW 的機器統稱控制器。最常見的就是鍵盤，其他還有數位打擊版、電子鼓、吉他合成器、吹管控制器等。
  • A/D：Analog to Digital。將類比訊號轉換成數位訊號的過程，此過程必須經過取樣 (Sampling) 的動作。ADC (Audio to Digital Converter) 就是類比數位轉換器。
  • D/A：Digital to Analog。將數位訊號轉換成類比訊號的過程，此過程必須將數位資料還原成類比電壓訊號。DAC 就是數位類比轉換器。ADC、DAC 也是錄音介面 (Audio Recording Interface)、高階合成器等級的關鍵之一。同時也是各個廠商的關鍵技術之一。
  • Audio Recording Interface (錄音介面)：台灣叫錄音卡、或者錄音介面。簡單說就是比較高級的音效卡 (Pro Sound Card，國外稱呼，參見拙作 “Pro 的!” 一文)。基本上跟電腦的音效卡架構差不多，主要都是作 A/D、D/A 的轉換處理。只是錄音介面的 A/D、D/A 等級比較高，也會有比較多的Channel可以同步運作 (多軌錄音)，同時也會提供多種音響用的街頭標準，像是類比訊號的 XLR (加農砲、麥克風接頭) 街頭、RCA (梅花接頭)、1/4吋(6.3mm) 接頭；數位訊號的 S/P DIF 接頭。
  • Sequencer，編曲機。指可以編輯 MIDI 資料的工作環境。軟硬體均有，硬體早期像是 Roland MC 系列 、 Yamaha QX 系列等。軟體像是 Cubase、Nuendo、SONAR、Logic、Digital Performer、ProTools 等都包含這個部份。
  • Audio Processor：指 DAW 的數位聲音訊號處理器，這也是 DAW 等級好壞的指標，如同錄音界面的 DAC 和 ADC 一樣。
  • Mixer/Console (軟體)：混音器/控制台。也就是錄音室裡各個音軌的控制環境，是早期錄音是最貴的硬體設備。現在已經有軟體可以取代。
  • Mixer/Amp (功率放大)：外部類比訊號混音器和功率放大器 (後級)
  • Monitor：監聽系統。主要目的就是要呈現原汁原味、沒有經過渲染的聲音，提供音樂製作時所需要的品質正確性。
  • 外部類比訊號：樂器經由麥克風收音的訊號、或者直接 Line-in 的訊號。
  • 外部數位訊號：一般指的是經由 S/P DIF 送給錄音界面處理的數位訊號，他不會經過 ADC 或 DAC 等轉換器。
  • Digital Effects：數位效果器。如 EQ (equalization等化器)、Reverb (殘響) 、Chorus (和聲效果)、Filter (濾波器)、Delay (延遲)、Compressor (壓縮器)、Overdrive (過載) … 等。
  • Score，樂譜輸出環境。大多數的 DAW 並不是以此為主要功能。樂譜有另外專門的軟體，如 Sibelius、PowerTrack (For Guitar)、Overturn (五線譜)、Guitar Pro (吉他) …等。相關概念請參閱『樂譜排版 (Music Score System) 與音樂製作 (Music Producer System)』 一文的介紹。

DAW / 硬體 扮演的角色

看了一堆名詞，簡單說明一些實際的例子，目前市面上常見的軟硬體在上圖中的角色：

  • Reason 的角色：主要是當成 Sound Module (黃灰色部份)、簡易的 Music Workstation (Sequencer 很陽春、沒有很強大的 Mixer/Console 環境、沒有 Audio Processing 環境)
  • Nuendo/Cubase、Sonar 的角色：完整的 DAW 環境 (黃色部份)，包含很強大的 Sequencer、Mixer/Console、Audio Processing、VST、VSTi plug-in (虛擬樂器) 介面支援 (不是包含有那些, 雖然有一些產品會送一些音源)、樂譜輸出。只要再搭配灰色的部分，就可以架構基本的音樂工作環境了。
  • 鍵盤樂器 (Keyboard、Synthesizer)：目前的工作站等級的 KB，除了樂譜排版，大部分都會包含圖中所有的單元。只是依照不同的等級、產品定位 (使用需求) 、行銷策略，會針對不同單元做比重的調整，因此會看到產品線上會有不同型號的產品。相關概念可以參閱 鍵盤分類介紹 (一) 、鍵盤分類介紹 (二) 的說明。 (updated: 20091214)
  • 吉他效果器 (Guitar Effect Processor)：吉他使用的重點在於 Digital Effect 這個部份，不會包含 Sound Module 的部份，一般也不會 Sequencer 和 Score。但是因為需要用來做參數控制，所以還是會具備 MIDI Message 部分。這邊的 MIDI 訊號意思不是說可以把吉他彈奏的音符轉換成我們認知的音符高低 (五線譜)，而是用來控制效果器的各個參數使用。所以現在很多效果器都可以透過 USB ，然後在電腦上調音色、備份設定等功能。 (updated: 20091214)

大概先介紹到這邊，接下來介紹的是當實際工作時，圖中每一個部分的訊號流向，以及他們是如何工作的。
