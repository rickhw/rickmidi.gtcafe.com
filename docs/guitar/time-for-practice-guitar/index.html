如何安排練習 - 以上班族為例

很多朋友都會有同樣的問題：怎麼安排時間練習，特別是上班族。

這個問題，特別看過像是：『吉他基本功』、『吉他音階學習引導』、『吉他和弦聲位基本概念』、『小調音階』 … 等之後，可能就不知道該怎麼辦了。。。因為很多人下了班，在床上躺平的時間可能都不夠了。特別如果又是爆肝工程師。。。

我舉我自己的例子給大家參考，如果你即將、或者已經是一個上班族。

找到目標

安排的基本前提：

  1. 找到目標、2) 了解自己，之後才來做 3) 安排時間。

目標是很重要的，練琴不只是為了練琴，而是要有方向的練。

有一個故事是這樣的，一個旅人在城市招了一台計程車， 上了車之後，還沒跟司機說要去哪，司機就加足了油門，往前衝。 客人很疑惑的問司機：我都還沒跟你說目地是哪裡，你怎麼就走了呢？ 結果司機說：對啊，不過我的技術很不錯，可以開得很快！

這裡告訴我們：

  1. 目標很重要，而且目標要可以到達。
  2. 沒有目標，衝的再快、技術再好，還是沒有用。

一、找到目標

目標不要好高騖遠，要實際一點。

目標是什麼？ 簡單且具體的，例如要 Cover 某一個樂團、歌手的歌曲， 像是 GNR、Muse、Bon Jovi、Steve Vai、伊凡賽斯、五月天、蘇打綠等； 或者是 Cover 一些流行歌、西洋老歌、藍調、爵士 … 等。 這些都是具體的模仿、學習對像，每一個階段，可能會有不同的方向。 但是，基本原則，就是不要一次想要吞太多東西， 小方向的達成即可。

二、了解自己

有了目標，開始要準備安排時間來執行。 但是在此之前，建議要花一點時間『了解自己』。 從旁觀察自己，包含自己的生理、心理狀態；習性、惰性、以及內心的聲音 什麼樣的狀態，能夠專注做事，這些條件是什麼？

Always listen to yourself … It’s better to be wrong than simply to follow convention, – From Startup 2.0 工程師創業手冊

生理方面

例如很多人習慣工作前來一杯咖啡、一杯茶、一根菸； 一些樂手、歌手錄音前，或者上台前，會先來一杯酒； 一些工程師寫 Code 前，會先去蹲馬桶，或者去投個藍； 然後排除不必要的干擾因素，像是網路、電話、簡訊 … 等。 如果要創作，有人會禁食、或者放下樂器到街頭遊走，或者乾脆去旅行 … … 總之，了解讓自己在練習前，獲得最佳狀態的前置條件 (Pre-Condition) 當然也要告訴你身邊的朋友或者親密愛人，不然他們可能會覺得你是很難相處的怪咖。

林正如老師在征服琴海提過一些類似的建議，是很好的參考。

心理方面

『了解自己』還有一點很重要，就是要很慎重、很認真的思考一件事情： 練琴、或者說音樂這件事情，對你的人生很重要嗎？ 他在你的人生中，在你的心裡，是什麼位置？ 過客？一時興起？玩玩而已？要靠它吃飯？ 如果你覺得在人生中他是一件事情， 那麼就要認真的看待它，認真的看待『練習』這件事情。

三、安排時間

回到主題，以上班族為例來說，目標是：『每天下班練琴兩個小時』

我在『吉他基本功』最後有提到：時間管理 這就是其中主要的要點之一：重要性、優先權。

每個人對於事情的重要性和優先權定義不一樣。 很多人下班大概行程都是這樣：

  • 下班吃飯、洗澡
  • 陪家人、陪女友/男友、 ….
  • 運動、打球 …
  • 看電視/電影、吃小吃、跟朋友/同事去吃飯
  • 打電動、看球賽
  • 逛百貨、看八點檔、政論節目
  • 上 FB、上 PTT、看迷片 …

有些是必要的，但是有些事可以取捨的。 所以要務實點、有取捨、了解自己、知道什麼是自己想要的， 那一些可以精簡，哪一些可以降低頻率。 然後，為自己營造練琴的環境（跟學英文很像）。

用我個人的例子分享給大家，我曾經有一段時間是這樣子走過來的。 那時候的生活很單純，只有兩件事：工作和音樂。 上班就是工作，工作內容就是 IT、寫程式、思考。 下班就是音樂，包含練琴、教學、練團、表演、Live House。

還好我工作不用爆肝，每天六點半就走了， 所以我下班後可以完全做跟音樂有關係的事情， 為了增加收入，也是激勵自己持續前進，平常下班後在自己的工作室兼吉他課。 (我不想時間被樂器行綁住，所以不去樂器行上課)

那段時間，週一到週五， 通常晚餐吃只吃一個麵包，然後會先睡覺半小時， 暖指半小時、上課一小時、運動一小時、備課或者練琴一小時。 沒課的時候，睡醒就先去跑步，然後再練琴練兩小時。 我會細分練的內容，以及進度。內容參考 吉他基本功 的細節。

所以練琴時間的安排就是：

  • 平常上班日 (一 ~ 五) 我只練基本功、技巧、不練歌，但是天天聽歌。聽要練的歌或者喜歡的歌。
  • 週末的時間完全都在練歌、練團、編曲、還有寫 Blog。

這說起來很容易，做起來很困難。 因為你可能不像我，除了工作之外，時間都放在音樂上。 我很清楚我要的東西以及方向，很清楚這樣會有所失、有所得。 失就是我不會去跟朋友唱歌，也不會週末去到處遊玩， 所以朋友可能會覺我好像很忙，可能會讓一些朋友有所誤解； 得就是我在音樂上的成長，不管是技術還是理論，都有相當的幅度， 同時我也因此認識一些熱愛音樂的朋友。

所以安排時，要想想自己想要的是什麼， 要適度的調整，以及找到屬於自己接受的方式。 不要一下定太高目標，因為做不到，只會打擊自己。

四、激勵自己

找一個可以激勵自己的方法，可以是外在的人事物，或者自我的實踐。 那時候常常會這樣鼓勵自己： 我想買設備，例如效果器、或者錄音界面、琴 (包含吉他 or Keyboard) ， 但是不可以用上班的薪水買，所以就接很多學生教、接表演； 或者找一些很難的歌來挑戰，在過程中，就是很努力練琴， 很努力 k 一些音樂書、分析歌曲結構、寫 Blog 唬爛， 或者找設備的使用手冊 (效果器或者 keyboard 的手冊都有) 來 k …..

我每天都有睡前閱讀的習慣，通常睡覺前會在床上看書一到半小時， 大部分是看金庸武俠小說、程式相關的書籍 (到現在還是)。 但是有一段時間我是在電腦前面看設備的使用手冊， 所以才有 “數位器材、音樂製作專有名詞對照表” 的整理。 這些手冊包含 Keyboard、吉他效果器、 或者一音樂使相關的書 (搖滾音樂史、藍調故事、爵士音樂史) ….. 這些故事是我上課或者跟朋友唬爛的來源 XDD

通常一段時間之後，錢存夠了、歌大概也練好了、 手冊也 k 的很熟了，就買設備來玩。這就是我給自己激勵的方式。 基本的原則就是，付出和收獲是可以對照的。

結語

自從兩年前 (2013) 我換工作後，我就更忙了，碰琴時間相對少很多。 偶爾還是會練練團，有時候也會有以前的團員找我做場子， 或者上課的教會問我有沒空，可否回去教琴。 但是一個人體力時間都有限，所以我只能用其他方式保持我的熱忱。 像是偶爾去練團室走走、在 FB 的社團和朋友交換想法 ….

這篇可能沒講到啥如何安排練習的細節， 不過分享了我個人曾經的過去給大家做參考。

我把電子書公佈之後，透過 email 或者在 FB 社團，都有朋友問我類似的問題， 特別是這些朋友大部分是上班族，我想就分享我的經驗給大家參考。 希望對大家有所幫助。
