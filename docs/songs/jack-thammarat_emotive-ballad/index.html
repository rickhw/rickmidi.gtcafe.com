Jack Thammarat - Emotive Ballad

原曲是 Guthrie Govan (GG) 的作品，算是抒情小品，但我更喜歡 Jack Thammarat 即興的版本。

Emotive Ballad 這首曲子的原曲是 Guthrie Govan (GG) 的作品，算是抒情小品。Jack Thammarat (JT) 在測試 Seymour Duncan STK-S7 時，做了一些改編。以下是 JT 測試 Seymour Duncan STK-S7 演奏的版本：

我是有一點先入為主，因為第一次聽到的也是 JT 的版本。其實 JT 也是用 GG 主要的旋律做一些改動，放入他自己慣用的句子。GG 原曲有一些慣性的『裝飾音、炫技』，這種東西好不好聽就見仁見智了。

JT 在情緒的營造，不會像老美或者英國吉他手那麼奔放狂野，或者說很多花俏的炫技，相對來看比較傳統、內斂且深邃，但是情緒曲線的安排也是很有張力的，特別是 run phase (很快的那段) 還是把情緒拉到最高點。而且右手有非常很細膩的 Touching / Dynamic，不管是 finger or picking … 旋律上則的五聲音階為主軸做變化 (參閱 Falling in Love Again 的分析)。
