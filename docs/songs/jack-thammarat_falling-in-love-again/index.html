Jack Thammarat - Falling in Love Again

這首 Falling in Love Again 吉他演奏曲是由 2009 Guitar Idol Winner - Jack Thammarat (泰國人) 譜曲 & 演奏

這首 Falling in Love Again 吉他演奏曲是由 2009 Guitar Idol Winner - Jack Thammarat (泰國人) 譜曲 & 演奏. 沒有聽過的朋友, 就先聽聽看 Jack 本人的詮釋吧.

Guitar Idol 是英國辦的類似星光大道的選秀節目, 聚集來自全球各地的吉他高手較競! 很精采的音樂盛宴!

除了這首曲子, 他在 Guitar Idol 比賽上彈的曲子 One the way 也是很好聽的, 不過演奏技巧難度又更高了.

從 Jack Thammarat 個人的 Youtube 上不難發現, 他的風格受的藍調, Eric Johnson 影響很多. 他喜歡詮釋一些經典歌曲, 像是 Eric Johnson 的名曲 Cliff of Dover (Squier + Korg PX4D, 讚!!), John Petrucci - Glasgow Kiss, 等. 另外他也 Demo 了一些科技產品, 像是 Boss GT8 的音色, Boss eBand JS-8, Graphtech Ghost Pickup System .. 等.

歌曲分析

回到 Falling in Love Again 這首曲子上, 歌曲有三大段, 和弦進行如下:

調號: E 大調

Intro I
||: B/C# / / / | B/A / / / | B/C# / / / | B/A / / / :||


Intro II / Bridge 
|| F#m7 G#m7 B C#m | F#m7 G#m7 Amaj7 / ||

Verse
||: Emaj7sus2 / / / | B/A / / / | Emaj7sus2 / / / | B/A / / / |
| G#m7 / / / | F#m7 / / / | A/B / / / :||
| B/C# / / / | B/C# / / / ||

Chorus
|| A / B / | Emaj7 / / / | A / G#m / | C#m / / / |
| A / B / | E / B/C# / | A / / / | A/B / B / ||

演奏技巧

先來看看要練這首歌曲之前, 需要的吉他演奏技巧:

基本技巧部分:

  1. Melody 主架構: E 大調五聲音階
  2. 藍調音 (b3) 表情貫穿整曲, 表情以滑音, 搥勾, 等技巧點綴的方式為主
  3. 小三度滑音: Mi - Sol 這個滑音/Bending 在這首歌曲是很重要的點. 我認為是呼應主題的關鍵點, Timing 不對就不對味了.
  4. 右手悶音: 讓旋律跳動的主要來源

Style 技巧: 包含了 Blues, 還有一些掃弦的東西.

  1. 經典藍調 Licks: 應用在 Bridge
  2. 一弦三音搥勾速彈 + 大九 (Xmaj9) 掃弦
  3. 藍調 Lick (Eric Clapton) 用一弦三音的手法來彈

其他技巧:

  1. 八度音: 在 Intro II / Bridge 裡的主要技巧
  2. 16beat Shuffle: 16beat shuffle, 尤其是在副歌的時候, 16beat shuffle feeling 讓歌曲有雀躍感, 微微的喜悅感, 更符合標題的意境 (有經驗的人都知道啦 >///////<).
  3. Pick up 的使用時機: Jack 習慣在長音 Bending, 通常也是段落銜接時, 換不同的 PU, 營造歌曲的張力.
  4. 右手 hybrid picking: Verse 右手以這樣的技巧詮釋為主

類似的演奏技巧/編曲手法在 Jack 另一首曲子 - One the way 應用的更多.

歌曲和聲分析

歌曲是 E 大調, 但是開頭 Intro 用了以 B 為主的和弦進行:

 || B/C# / / / | B/A / / / | B/C# / / / | B/A / / / ||

歌曲開頭是五級, 這是很特別的. 除非歌曲是很明顯的調式, 不過這首聽起來除了使用藍調 (E Blues) 之外, 應該不是調式.

我猜測編曲時的意圖是: 利用 B (B D# F#) 來取代 Emaj7sus2 (E D# F#), 然後再把 B 的修飾和弦 Badd9 和屬七 B7 改成 Slash Chord, 形成 B/C# (Badd9) 和 B/A (B7) 這樣的進行.

這樣的效果保留了 Emaj7sus2 的聲響, 但是 Bass line 卻是 4 - 6, 很 Fusion 的編曲手法. 為什麼我會想到可能是 Emaj7sus2 這個和弦的關係? 請繼續往下看.

接下來 Intro 有一段主題, 就是歌曲當中八度音的十六分音符. 這段在 E 大調的進行級數是:

| 2m7 3m7 5 6m7 | 2m7 3m7 4M7 / |

差不多一拍一個和弦. 拍子很密集, 以十六分音符為單位, 加上一點 funk 的八度音手法, 聽起來很輕快 ~~

主歌 Verse 的進行: 出現 Emaj7sus2 這這樣的和弦, 讓我聯想到歌曲一開始的和弦 B/A, B/C#. 因為聲響很接近, 都有 B 和弦的組成音, 吉他的指型也很像.

以一般作曲的流程來看, 可能會先寫出副歌 (hook) -> 主歌 -> 完成前奏間奏尾奏. 前奏間奏尾奏在編曲階段, 為了要呼應主題, 經常會取材自主歌 or 副歌, 然後加以變化. 所以依照這樣的邏輯推算, 前奏的和弦進行有可能就是取材自主歌或副歌的某個部份. 這邊我假設的是取材自主歌的前四小節. 比較如下:

  • 前奏: || B/C# / / / | B/A / / / | B/C# / / / | B/A / / / ||
  • 主歌: || Emaj7sus2 / / / | B/A / / / | Emaj7sus2 / / / | B/A / / / |

接下來是歌曲當中出現很多 Slash Chord, 像是 B/A, A/B, B/C#, 基本上我都把他們當成是一種同組成音異名的和弦 (請參閱: 關係和弦), B/A = B7, A/B = Aadd9, B/C# = Badd9. 不過都把那些七度音, 裝飾音, 從原本是點綴的功能, 搬到 Bass Line 去了, 這樣等於加強了七度音, 九度音等. 類似的編曲手法也出現在副歌 Chorus 的部份, 而且還多了 Bass Line 的層次感:

副歌出現以下的進行:

 | A/B / B / | ==> | 4/5 / 5 / |

可以看成是:

 | Aadd9 / B / | ==> | 4add9 / 5 / |

但是彈 Aadd9 跟 A/B 聽到的層次感是不一樣的, 前者聽起來只是修飾 A, 後者聽起來就是有點類似搶拍的手法, 不過搶拍的是 Bass, 因為五級 Bass 提前出來.

跟技巧一樣, 同樣的概念, 在 Jack 另一首成名曲 On the way 也都出現很多次.

另外副歌 (Chorus) 的第一個和弦是四級, 四級有讓人往前進, 向前看的感覺, 可以說是呼應歌曲主題. 在一般教會音樂常用 I - IV 進行, 通常要寫出比較有激勵人心, 比較熱血, 情緒向上的音樂, 可以使用這樣的和弦進行.

創作分析

想要寫出旋律容易聆聽, 平易近人, 有旋律性, 有記憶點 (hook), 但是又不會感覺陳腔濫調, 像這樣的歌曲, 其實是有一些技巧的. 從這些年來的一些當代音樂, 像是 Fusion 註 這種 “具有融合性” 的音樂, 最常見的就是音樂三大核心概念 (節奏, 旋律, 和聲) 出發, 以五聲音階為旋律主架構, 配上爵士和聲, 加上搖滾的節奏:

  1. 旋律 - 五聲音階 (Pentatonic Scale): 五聲音階具備好聽, 有旋律性的特色, 配合設計過的和聲架構, 透過五聲音階與和弦交互而成調式的關係, 讓旋律更有特色, 聆聽者也不容易感到膩. 同時五聲音階也常常會和藍調音階放在一起應用, 藍調音階又具備讓音樂呈現極具張力的特性, 所以研究如何好好應用五聲音階, 是作曲 & 即興很不錯的一個方向. 這點在一些當代 Fusion 大師的作品身上可以驗證到.
  2. 和聲 - 爵士和聲 (Jazz Harmony): 五聲音階好聽, 有旋律性, 但是和聲卻很弱, 所以利用合聲功能很豐富的爵士和聲理論來架構音樂結構, 控制情緒溫度/特色, 建立段落的張力. 這類的技巧很多, 常見的像是 Modal Interchange, Secondary Dominant, TSD 和聲代換, 關係和弦… 等技巧, 增加整個音樂的張力, 情緒顏色.
  3. 節奏 - 搖滾的節奏 (Rock Rhythm): 讓音樂動起來, 讓聆聽者感覺音樂的活力. 這邊搖滾的包含了所有搖滾的旁支風格, Rock / Metal / Funk … etc, 甚至是 Blues, Latin, …. etc.

這樣的組合, 讓音樂除了好聽, 有活力, 而且層次分明, 豐富. 這樣的作曲/編曲方式, 是我這些年來在聆聽/分析一些歌曲時的心得, 現在好聽的音樂大多具備這些特性. Jack Thammarat 比賽上彈的曲子 One the way 也是也同時具備這些條件.

除了卡農進行之外, 我個人覺的, 這種 Fusion 註 也是近代音樂作曲演進上的經典.

關於 Fusion Style

關於 Fusion 這種音樂風格 (Style), 一般中文翻譯成 “融合爵士樂”, 這種風格其實很難定義, “融合” 簡單說就是兩種以上風格的混合, 像是 Jazz-Rock> 這樣的組合可以叫 Fusion, Blues-Rock , Jazz-Blues … 等都可以算是 Fusion.

Fusion 從什麼時候已經很難確定, 大概是 70' 年代的時候發展出來的. 爵士大師 Miles Davis 的 “In a Silent Way” 專輯可以當作一個參考點.

這樣的音樂, 在當年這就像是, 喝了三十年的黑咖啡, 有一天突然發現, 在黑咖啡裡加入牛奶, 味道很特別 .. 所以 … 從 Fusion 就誕生了, 也因此, 中文翻譯通常會翻成 “融合爵士樂”.

不過這樣的 “融合” 在現代的音樂中, 似乎聽起來是很理所當然的. 就像今天喝的咖啡一樣.

不過人類文化的發展大概是有一個週期性的, 簡單的東西會往複雜發展, 發展到一定程度就會打散或者爆炸, 又開始漸漸回歸到簡單/單純, 或者發展出新的東西. 簡單/單純或新的東西一段時間又開始漸漸複雜, 如此週而復始的循環.

Fusion 是一種複雜度很高的音樂, 發展到最後, 又開始回歸自然, 所以 2000 年左右, 一些原本玩 Jazz/Fusion 的大咖, 都跑回去玩 Blues 了, 而且是比較純的藍調. 同時在台灣, 陳綺貞/張懸這樣清新簡單的風格, 又給當時音樂市場一個新的體驗. 其實不是啥新的體驗, 只是又開始循環了.

同樣的東西在 IT 領域也是一樣, 高複雜度的程式語言發展到一定程度 (ex: C/C++), 就會有人開發出號稱 “很簡單, 容易學習” 的語言, Java 就是一個例子. 但是 Java 發展了 15 年來, 早就已經忘了初衷了, 現在的 Java 是又複雜, 又難學, 又難寫, 又大又肥又噁心, 加上一堆噁心的符號混在程式碼裡 …. 所以一堆新的語言又出來了, 標榜 “Easy learning, Easy programming” …. 然後 ……… 我又開始碎碎念了, 聽歌吧 XDD
