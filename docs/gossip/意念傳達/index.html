意念傳達

學樂器大致上分成兩個部分：第一部分是學習樂器的駕馭、第二部分是學習利用樂器來表達東西。

學吉他 (或者說學樂器) 大致上分成兩個部分：

一個是學習吉他這個樂器的駕馭。像是手指跟弦的感覺，手指的獨立性，指版位置熟悉，右手與弦的 touch (dynamic)，左手壓弦的感覺，吉他的聲音特性 … 等。跟及他有直接關係的 “技術”。這些都是必須要靠後天的練習，熟悉才能夠慢慢的熟悉。

第二部分是學習利用 “吉他” 這個樂器來表達東西。簡單說，就是一種思想表現，意念的傳達。

學過 Java 語言的人也許看過一本書：Thinking in Java 。作者在開始的時候提到，程式語言本身的概念：

"But computers are not so much machines as they are mind amplification tools ("bicycles for the mind," as Steve Jobs is fond of saying) and a different kind of expressive medium. As a result, the tools are beginning to look less like machines and more like parts of our minds, and also like other forms of expression such as writing, painting, sculpture, animation, and filmmaking. Object-oriented programming (OOP) is part of this movement toward using the computer as an expressive medium. "

摘錄自：Thinking in Java，3rd
  • Thinking in Java
  • Object-oriented programming

中文意思大概就是說：電腦是一種意念表達的一種媒介 / 工具，有點像是寫作、繪圖、雕刻、動畫、電影 … 等表達形式。而 OOP (Object-oriented programming 物件導向程式設計) 就是一種意念表達的方式。

大部分學習程式語言也都會有兩個階段，一是熟悉程式語言本身的基本特性 / 語法 / 表達方式 / 關鍵字 (語言特性) … 等。第二部分就是學習利用語言特性來表達東西。Java / C++ 是利用 OOP 的概念表達意念；RoR (Ruby on Rails) 是利用 meta-programming 的方式來表達；Functional Programming 則是現今面對複雜的多核心時，從新找到定位的技巧 …

Functional Programming: 因為多核心編程關係，又重新崛起的編程技術。目前常見的主流語言 Erlang、Scala (JVM)、F#、Haskell、Javascript

吉他也是一種表達方式 / 工具，透過吉他來表達音樂，傳達一些想法 / 意念。

音樂這種藝術也是表達意念的方式之一，而吉他、鋼琴、小提琴、鼓、Obeo … 等，都是媒介之一。人類之所以與其他萬物不同，除了人會想明天，我想，意念的傳達的方式是很重要的因素之一。
