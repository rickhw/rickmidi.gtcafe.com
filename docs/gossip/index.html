聊音樂

關於音樂的想法 ….
簡單、複雜

真實的人生，是複雜的，但大部份的人所期望的，或者說習慣的，是簡單的，也就是舒適的環境。習慣一家公司的制度、習慣在家裡、習慣每個月領一包薪水、習慣住在什麼都有的房子或地區、習慣每天見到某一些人、習慣每天走一樣的路、聽某一些音樂 … 而複雜的環境，卻是每天都要面對的。

對談

最近因緣際會和音樂圈的老前輩閒話了一番.

保護耳朵

現在隨身聆聽設備很方便，現在智慧型手機，都可以播放音樂，使的不管在路上，還是辦公室裡，很常會看到大家帶著耳機聽著自己的音樂。樂手是真的需要大量聆聽的，一方面吸收新資訊，一方面聆聽好的作品，對於提升音樂的素養是非常重要的。因為『 CD是最好的老師』… 不過 …

練習的層次

在教學時，有時候同學問到關於什麼樣的狀況才叫做 “練好”。這是個好問題，尤其打算練一些 “經典歌曲” 的時候。

關於速度

在人類的世界裡，速度永遠是一個話題，不管在藝術、科技、生活、體育 …等各種領域。
